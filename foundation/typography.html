Foundation

Typography

본문 글꼴은 Pretendard. 21개 역할(Header 3 → Caption 2)을 Bold · SemiBold · Medium · Regular 4개 weight로 운용합니다. line-height는 배수, letter-spacing은 em으로 표기하며 Figma 변수와 1:1 대응합니다.

Font Family

국문·영문·숫자 모두 Pretendard 한 벌로 표기합니다.

프리텐다드 Pretendard 0123456789

Pretendard Regular400

Type Scale

각 역할의 size / line-height / letter-spacing 명세와 4 weight 미리보기.

사용 방법

역할 클래스(.t-*)와 weight 유틸리티(.fw-*)를 조합합니다.

<h1 class="t-display-2 fw-bold">제목</h1> <p class="t-body-1-reading">본문 (읽기용 행간)</p> <span class="t-caption-1 fw-medium fg-muted">캡션</span>
