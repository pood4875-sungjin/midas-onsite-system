Shadow

엘리베이션 4단계. 베이스 컬러는 #131927 이며 각 단계는 2겹의 drop-shadow로 구성됩니다. 숫자가 클수록 표면이 더 높이 떠 있는 느낌을 줍니다.

⚠ 이 토큰은 시안 첨부 후 변경될 수 있습니다. 현재 값은 Figma(node 9:120)에서 추출한 ver0.1 기준입니다.

명세

각 단계는 2겹 drop-shadow. 투명도는 8-digit hex로 표기.

토큰용도레이어
--shadow-10버튼 · 칩 · 작은 카드 hover0 4 4 -2 #13192714 · 0 2 4 -2 #1319271F
--shadow-30카드 · 팝오버 · 드롭다운0 6 32 -4 #1319271A · 0 3 14 -6 #1319271F
--shadow-50토스트 · 플로팅 패널0 18 64 -4 #1319271F · 0 10 22 -6 #1319271F
--shadow-70모달 · 다이얼로그0 40 100 -4 #13192724 · 0 20 32 -6 #1319271F

사용 방법

.card { box-shadow: var(--shadow-30); } .toast { box-shadow: var(--shadow-50); } .modal { box-shadow: var(--shadow-70); }
