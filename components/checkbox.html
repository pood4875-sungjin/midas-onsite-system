Checkbox

선택·해제·부분선택(indeterminate) 3가지 선택 상태 × 활성·비활성 = 6종. 선택색은 MIDAS Emerald, 박스 보더·배경은 시맨틱 토큰으로 라이트·다크에 자동 대응합니다.

◍ 이 컴포넌트는 임시(provisional)입니다. 색·치수는 토큰 체계 기준으로 구현했으며, MIDAS Figma 원본 확정 후 보정됩니다.

부모-자식 (Indeterminate)

상위 체크박스로 전체 선택/해제. 일부만 선택되면 indeterminate 로 표시됩니다.

사용 방법

<label class="checkbox"> <input class="checkbox__input" type="checkbox" /> <span class="checkbox__box" aria-hidden="true"> <svg class="checkbox__check" viewBox="0 0 24 24">…</svg> <span class="checkbox__dash"></span> </span> <span class="checkbox__label">동의합니다</span> </label> <!-- indeterminate 는 JS: input.indeterminate = true -->
