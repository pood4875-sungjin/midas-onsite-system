Input

라벨·도움말·에러를 포함한 텍스트 필드. 상태(Default·Focus·Error·Success·Disabled)와 아이콘·textarea 변형을 제공하며, 인라인 검증 패턴을 기본으로 합니다.

◍ 이 컴포넌트는 임시(provisional)입니다. 색·치수는 토큰 체계 기준으로 구현했으며, MIDAS Figma 원본 확정 후 보정됩니다.

States

Default · Focus · Error · Success · Disabled · Read-only. 라벨/도움말은 항상 시맨틱 색.

아이콘 · 크기

앞/뒤 affix로 아이콘·버튼 배치. 기본 44px, SM 36px.

Textarea

여러 줄 입력. 세로 리사이즈만 허용.

최대 500자.

인라인 검증 데모

필드를 벗어날 때(blur) 검증합니다. 빈 값이면 에러, 올바르면 success. (ux-patterns 폼 인라인 검증 근거)

포커스를 벗어나면 검증합니다. 올바른 이메일을 입력하세요. 좋습니다!
