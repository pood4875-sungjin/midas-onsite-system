Components · Navigation

Tab

텍스트 탭 1타입. 대시보드 서브메뉴나 페이지 내 섹션 전환에 사용합니다. 선택된 탭은 하단 인디케이터(2px)와 SemiBold로 구분되며, 라이트·다크 모드에 자동 대응합니다.

States

Default · Hover · Selected · Disabled. 텍스트 색과 하단 인디케이터로 상태를 표현합니다.

Interactive

탭을 클릭하면 패널이 전환됩니다. 좌우 방향키로 이동할 수 있습니다(WAI-ARIA Tabs 패턴).

대시보드 — 현장뷰어, 시공단계 테이블, 계측 현황 요약을 한 화면에서 봅니다.

결과 — 단면별 해석 결과를 위험도 순으로 확인합니다.

계측그래프 — 변위·응력 시계열 그래프를 비교합니다.

보고서 — 정기·주간 보고서를 생성하고 공유합니다.
